# InternSpark Data Analytics Internship

## Overview

This repository contains three data analytics projects completed as part of the InternSpark Data Analytics Internship.

The projects focus on developing practical skills in:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Data Visualization
* Business Reporting
* Insight Generation

All projects were completed using Python and Google Colab.

---

## Projects Completed

### 1. Customer Behavior Analysis

Analyzed customer purchase behavior using an e-commerce dataset.

Key areas explored:

* Customer demographics
* Product category preferences
* Payment methods
* Customer churn analysis
* RFM segmentation

**Skills Demonstrated**

* Customer Analytics
* Churn Analysis
* RFM Segmentation
* Business Insight Generation

---

### 2. Sales Performance Analysis

Analyzed retail sales performance to identify revenue and profitability trends.

Key areas explored:

* Sales trends
* Category-wise performance
* Regional performance
* Profit analysis
* Business recommendations

**Skills Demonstrated**

* Sales Analytics
* Revenue Analysis
* Data Visualization
* Performance Reporting

---

### 3. Website Traffic Analysis

Analyzed website traffic data from a music marketing platform.

Key areas explored:

* Traffic distribution
* User engagement
* Country-wise activity
* Artist popularity
* Trend analysis

**Skills Demonstrated**

* Traffic Analytics
* Trend Analysis
* User Behavior Analysis
* Data Storytelling
